## 一、脚本一：B站搜索结果批量保存脚本
### （一）脚本定位
本脚本为B站搜索结果页专属高效工具，核心定位是**替代手动复制操作，实现搜索结果页视频链接及相关信息的自动化、批量化采集与保存**，解决用户在B站搜索特定内容后，需要逐个点击视频、复制链接、记录信息的繁琐问题，一键完成批量数据抓取，满足快速归档、批量整理搜索结果的核心需求。

### （二）核心功能详解
1. **全量搜索结果抓取**
脚本可自动识别B站搜索结果页的所有视频卡片内容，无论单页展示20条、40条还是更多视频，均可实现无遗漏抓取，精准提取每一条视频的核心信息，包括视频标题、UP主名称、BV号、视频播放页面直达链接、视频发布时间、播放量、弹幕量、视频时长等完整信息，避免手动筛选出现的信息遗漏。

2. **多页自动批量抓取**
支持自定义设置抓取页数，用户可根据自身需求，自由选择抓取搜索结果的前5页、10页、20页甚至更多页面，脚本会自动模拟翻页操作，无需用户手动点击下一页，逐页完成视频信息与链接抓取，全程自动化运行，大幅减少手动操作步骤，适用于大量搜索结果的批量整理场景。

3. **智能去重处理**
针对搜索结果中可能出现的重复视频、重复链接，脚本内置智能去重机制，通过识别视频BV号、播放链接等唯一标识，自动过滤重复数据，确保最终保存的视频列表无重复内容，无需用户后期手动筛选删除，提升资源整理的精准度与整洁度。

4. **多格式保存与导出**
抓取完成后，支持多种保存导出方式，满足不同用户的使用习惯：一是一键复制所有视频链接至剪贴板，可直接粘贴至文档、表格或下载工具；二是导出为TXT文本文件，文件内清晰罗列每条视频的标题、BV号、播放链接，便于本地长期归档；三是支持导出为CSV格式，可直接用Excel打开，方便进行后期分类、筛选、编辑操作。

5. **实时预览与筛选**
抓取过程中支持实时预览已抓取的视频列表，用户可直观查看抓取进度与视频信息；同时支持临时筛选功能，可根据视频时长、播放量、发布时间进行初步筛选，剔除不符合需求的视频，再进行最终保存，提升资源整理的针对性。

6. **暂停与终止操作**
脚本运行过程中支持随时暂停、终止抓取，若用户中途需要修改抓取页数、停止操作，无需关闭页面，一键即可控制脚本运行，操作灵活度高，避免误操作导致的重复抓取。

### （三）适用场景
1. B站搜索特定主题、关键词、UP主作品后，批量保存所有相关视频链接，用于个人素材库整理、学习资料归档；
2. 自媒体创作者批量收集选题素材，保存同类型视频链接，方便后续参考、剪辑；
3. 需要批量获取某一关键词下的视频资源，为后续批量下载做前期准备；
4. 网课、学习类视频搜索结果批量保存，按顺序归档学习内容，避免逐个收藏的繁琐。

### （四）操作流程
1. 安装脚本后，打开B站网页端，进入任意关键词的搜索结果页面；
2. 点击页面左上角悬浮的脚本控制面板，设置需要抓取的搜索结果页数；
3. 点击“开始抓取”按钮，脚本自动逐页识别、采集视频信息，实时显示抓取进度；
4. 抓取完成后，可预览视频列表，进行简单筛选；
5. 选择复制链接、导出TXT、导出CSV等方式，完成批量保存。

### （五）适配范围
完美适配B站网页端所有搜索结果页面，包括综合搜索、视频搜索、番剧搜索、课程搜索等各类搜索场景，兼容不同排序方式（综合排序、播放量排序、最新发布排序）下的搜索结果，无论电脑端宽屏、窄屏显示，均能精准识别视频卡片，不影响抓取效果。

### （六）优势亮点
1. 全自动运行，无需人工干预，大幅节省操作时间；
2. 信息提取全面，核心视频信息一站式采集，无需额外记录；
3. 智能去重，避免重复资源，提升整理效率；
4. 多格式导出，适配不同归档、使用需求；
5. 轻量化运行，不占用电脑内存，不影响B站页面正常浏览。

## 二、脚本二：Alt键一键获取当前视频链接脚本
### （一）脚本定位
本脚本为B站视频播放页快捷操作工具，核心定位是**简化视频链接获取流程，通过快捷键一键触发，快速提取当前播放视频的可直接调用下载的有效链接**，摒弃手动点击复制、跳转页面查找链接的繁琐步骤，实现一键提取、一键复制，无缝对接B站视频下载工具，满足快速下载单个视频的需求。

### （二）核心功能详解
1. **快捷键一键触发**
采用全局Alt键作为触发快捷键，无需点击任何按钮、无需打开任何菜单，在B站视频播放页面，无论视频处于播放、暂停、全屏（非网页全屏）状态，只需按下键盘Alt键，即刻触发链接提取功能，操作极简、响应迅速，不干扰正常视频观看。

2. **精准提取有效下载链接**
区别于普通的视频播放页面链接，脚本可直接提取适合下载工具调用的有效视频链接，包括视频原生播放链接、BV号专属链接、无跳转直链等，避免提取无效链接、短链接，确保下载工具可直接识别、解析，无需二次转换链接格式。

3. **自动复制至剪贴板**
提取链接成功后，脚本自动将有效视频链接复制到电脑剪贴板，无需手动选中、复制，弹窗实时提示“链接提取并复制成功”，用户可直接打开下载工具，粘贴链接即可开始下载，操作流程一步到位。

4. **全类型视频适配**
适配B站所有类型的视频内容，包括UP主原创投稿视频、番剧动漫、课程网课、直播回放、合集视频、分P视频等，无论单P视频还是多P视频，均可精准提取对应视频的专属链接，分P视频可单独提取每一P的链接，无适配盲区。

5. **无界面干扰设计**
脚本采用无悬浮窗、无多余控件的极简设计，全程不显示任何多余界面，不遮挡视频播放画面、不影响弹幕观看、不干扰页面正常操作，仅在触发快捷键后弹出短暂提示弹窗，用完即关闭，保持B站页面原生浏览体验。

6. **链接校验功能**
内置链接自动校验机制，提取链接后自动检测链接有效性，若出现链接提取失败、无效链接的情况，立即弹窗提示“链接提取失败，请重新触发”，避免复制无效链接导致下载失败，提升使用可靠性。

### （三）适用场景
1. 观看B站视频时，需要快速下载当前视频，无需手动查找复制链接，一键提取即可调用下载工具；
2. 批量下载单个UP主多个视频时，逐个打开视频，按Alt键快速复制链接，批量存入下载列表；
3. 看到优质网课、学习视频、素材视频，快速保存链接，避免退出播放页后找不到视频；
4. 合集、分P视频，单独提取某一P的视频链接，精准下载所需内容，无需下载整个合集。

### （四）操作流程
1. 安装脚本后，打开B站任意视频的播放页面；
2. 正常观看视频，需要获取链接时，按下键盘Alt键；
3. 脚本瞬间提取当前视频有效下载链接，自动复制至剪贴板，并弹出提示窗口；
4. 打开B站视频下载工具，直接粘贴链接，即可开始下载。

### （五）适配范围
适配B站网页端所有视频播放页面，兼容普通播放页、全屏播放页、合集播放页、番剧专属播放页、课程播放页等，支持Windows、Mac系统的所有主流浏览器，无论键盘布局差异，Alt键均可正常触发，无按键冲突问题。

### （六）优势亮点
1. 纯快捷键操作，零界面操作，极致简洁；
2. 提取链接精准有效，直接适配下载工具；
3. 无页面干扰，不影响正常视频观看体验；
4. 响应速度极快，按下按键即刻完成提取复制；
5. 全类型视频适配，无使用场景限制。

## 三、脚本三：多关键词批量拆分+智能爬取脚本
### （一）脚本定位
本脚本为B站高效批量资源获取工具，核心定位是**实现多关键词批量导入、自动拆分、逐次智能爬取，同时自定义爬取页数与视频时长筛选**，解决用户需要多次输入关键词、反复设置筛选条件、手动控制下载范围的繁琐问题，一站式完成多关键词视频资源的批量采集，满足大规模素材收集、精准筛选视频的深度需求。

### （二）核心功能详解
1. **多关键词批量导入与自动拆分**
支持批量输入多个关键词，用户可将所有需要搜索的关键词一次性粘贴至脚本输入框，关键词之间可通过换行、逗号、顿号、空格等多种分隔符区分，脚本自动识别并拆分每一个独立关键词，无需逐个手动输入、逐个搜索，大幅提升多主题内容检索的效率。

2. **逐关键词自动搜索爬取**
关键词拆分完成后，脚本按照输入顺序，自动依次调用B站搜索功能，进入每个关键词的搜索结果页，按照预设参数自动爬取视频信息与链接，全程无需人工干预、无需切换页面、无需重复操作，完成一个关键词的爬取后，自动跳转至下一个关键词，直至所有关键词爬取完毕。

3. **自定义爬取页数设置**
针对每一组关键词，支持统一自定义设置爬取页数，用户可根据关键词的搜索结果数量，自由设定爬取前N页内容，可灵活设置1页、5页、10页甚至更多，既可以精准获取少量核心资源，也可以大规模采集全面资源，适配不同的资源获取需求。

4. **视频时长精准筛选**
内置视频时长智能筛选机制，用户可提前设置视频时长范围，包括最短时长、最长时长，脚本在爬取过程中，自动过滤不符合时长要求的视频，只保留指定时长范围内的资源，例如筛选5-15分钟的中短视频、30分钟以上的长视频、10分钟以内的素材片段等，避免爬取大量无关时长的视频，减少后期筛选工作量。

5. **批量数据汇总与统一导出**
所有关键词爬取完成后，脚本自动汇总全部有效视频数据，整合为统一的资源列表，清晰标注每个视频对应的关键词、视频标题、时长、播放链接，支持一键复制所有链接、批量导出为TXT/CSV文件，便于统一管理、批量下载，实现多关键词资源的一站式归档。

6. **异常处理与断点续爬**
针对爬取过程中可能出现的页面加载失败、关键词无搜索结果、网络延迟等问题，脚本内置异常处理机制，自动跳过异常页面、无结果关键词，继续执行后续爬取任务，同时支持断点续爬功能，若中途关闭页面或停止脚本，再次启动可从中断的关键词继续爬取，避免重复操作。

7. **爬取进度实时监控**
脚本控制面板实时显示当前爬取关键词、已完成数量、剩余数量、当前爬取页数、已抓取视频数量等信息，用户可直观掌握整体爬取进度，随时查看爬取状态，便于把控任务进度。

### （三）适用场景
1. 自媒体创作者、剪辑师批量收集多主题、多类型视频素材，一次性导入所有选题关键词，自动爬取符合时长要求的素材链接；
2. 学生、职场人士批量整理多主题学习资料、课程视频、行业干货视频，精准筛选时长合适的学习内容；
3. 资源整理爱好者针对多个兴趣领域，批量获取相关视频资源，建立专属资源库；
4. 需要大规模采集B站某一领域、多类关键词的视频数据，用于数据分析、选题调研、内容盘点等；
5. 批量下载多主题视频，提前通过时长、页数筛选，精准获取所需资源，避免无效下载。

### （四）操作流程
1. 安装脚本后，打开B站网页端，调出脚本控制面板；
2. 在关键词输入框中，批量粘贴所有需要搜索的关键词，用分隔符分开；
3. 设置统一的爬取页数、视频时长范围（最短时长、最长时长）；
4. 点击“开始批量爬取”按钮，脚本自动拆分关键词，逐次搜索、爬取；
5. 实时查看爬取进度，等待所有关键词爬取完成；
6. 预览汇总后的视频列表，确认无误后，选择复制链接或导出文件，完成资源保存。

### （五）适配范围
适配B站网页端全搜索场景，支持所有类型关键词的搜索爬取，兼容各类关键词组合、长关键词、短关键词，适配不同浏览器的页面加载模式，无论页面加载速度快慢，脚本均可智能等待加载完成后再爬取，确保数据抓取完整。

### （六）优势亮点
1. 多关键词一站式处理，无需逐次操作，效率提升数倍；
2. 时长+页数双重精准筛选，资源获取更具针对性；
3. 全自动批量爬取，解放双手，适合大规模资源采集；
4. 数据统一汇总，便于后期管理与下载；
5. 异常处理完善，爬取稳定性高，减少失误与重复操作。

## 四、三款脚本通用注意事项
1. 三款脚本均为油猴脚本，需在安装Tampermonkey（油猴）/Violentmonkey（暴力猴）扩展的浏览器中运行，仅支持B站**电脑网页端**，不支持B站移动端、客户端；
2. 脚本全程本地运行，所有数据抓取、处理、保存均在用户本地浏览器完成，无任何数据上传至第三方服务器，不收集用户隐私信息、浏览记录，使用安全无风险；
3. 脚本仅提取B站公开可浏览的视频链接与信息，不破解视频版权、不绕过视频加密、不获取付费视频、会员专属视频内容，仅用于个人学习、素材整理，请勿用于商业用途、侵权用途；
4. 使用过程中，若B站页面更新、布局改版，可能导致脚本暂时失效，届时可根据页面变化调整脚本适配逻辑；
5. 批量爬取时，建议合理设置爬取页数、间隔时间，避免短时间内大量请求导致B站页面加载受限，保障脚本稳定运行；
6. 三款脚本可同时安装、同时运行，互不冲突，可根据使用需求自由开启、关闭对应脚本。
