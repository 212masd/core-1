### 快速关闭侧边栏（左边那栏）
**最快：按一下 `Ctrl + B`（Windows/Linux）或 `Cmd + B`（Mac）**
- **B = Toggle (Sidebar)** → 一键**隐藏/显示**整个侧边栏
- 按一次：侧边栏**完全消失**
- 再按一次：侧边栏**恢复**


### 首字母记忆（你要的规则）
- **B** = **B**ar / **B**rowse（侧边栏、浏览面板）
- 记住：**B = 边栏开关**

### 其他常用栏速关（一起记）
- **底部面板（终端/问题）**：`Ctrl + J` → **J = Panel**

- **右侧小面板（大纲/扩展）**：`Ctrl + Alt + B`
- **全屏写作（无任何栏）**：`Ctrl + K Z` → **Z = Zen**（禅模式）

# 一、侧边栏 & 界面控制
- **Ctrl + B**

  首字母：**B** = Bar（边栏）

- **Ctrl + J**
  功能：显示 / 隐藏底部面板（终端、问题）

- **Ctrl + Alt + B**
  功能：显示 / 隐藏右侧栏（大纲、预览）
