# 二、文件操作（真实可用）
- `Ctrl + N`
  新建空白文件

- `Ctrl + S`
  保存当前文件

- `Ctrl + Shift + S`
  另存为

- `Ctrl + W`
  关闭当前文件

- `Ctrl + Tab`
  在打开的文件之间切换

- `Ctrl + P`
  快速查找并打开文件

- `F2`
  重命名文件 / 文件夹

- `Delete`
  删除文件 / 文件夹

---

# 三、文件树专用（必须先选中文件树）
这些**必须先点一下左侧文件列表**才生效：

- `↑ / ↓`
  上下选择文件/文件夹

- `→`
  展开文件夹

- `←`
  折叠文件夹

- `Enter`
  打开文件 / 进入文件夹

- `Ctrl + Shift + N`
  在**当前选中文件夹下新建文件夹**

---

# 四、文件夹折叠/展开（真的能实现）
**不是同时按！是顺序按：**

1. 先按 `Ctrl + K` → 松开
2. 再按 `0`
→ **全部折叠**

1. 先按 `Ctrl + K` → 松开
2. 再按 `J`
→ **全部展开**

---

# 五、编辑文字（写作用）
- `Ctrl + C`
  复制

- `Ctrl + X`
  剪切

- `Ctrl + V`
  粘贴

- `Ctrl + Z`
  撤销

- `Ctrl + Y`
  重做（恢复撤销）

- `Ctrl + F`
  当前文件内查找

- `Ctrl + D`
  选中下一个相同单词

- `Ctrl + Enter`
  下方插入空行

- `Ctrl + Shift + Enter`
  上方插入空行

---

# 六、代码/文本折叠（写长文很有用）
- `Ctrl + Shift + [`
  折叠当前段落

- `Ctrl + Shift + ]`
  展开当前段落

---

# 七、视图模式
- `Ctrl + K Z`
  禅模式（全屏无干扰写作）
  按两次 `Esc` 退出

---

11. 快速打开文件：`Ctrl+P`
12. 查找文字：`Ctrl+F`
13. 全局搜索：`Ctrl+Shift+F`
14. 复制：`Ctrl+C`
15. 剪切：`Ctrl+X`
16. 粘贴：`Ctrl+V`
17. 撤销：`Ctrl+Z`
18. 重做：`Ctrl+Y`
