# 真实可用版：在当前目录新建文件、文件夹（无编造）
## 前提
必须先让**光标聚焦在左侧文件树**里，否则快捷键无效。

聚焦文件树：
**Ctrl + Shift + E**
（E = Explorer，真的官方快捷键）

---

# 1. 在当前目录下新建文件夹（真实快捷键）
1. 用 ↑ ↓ 选中你要新建在**哪个文件夹下面**
2. 直接按：
**F2 旁边那个键：`]` 右边的 `\` 键？不，是：**

## 真正官方唯一方式：
选中文件夹后，直接按：
### **Ctrl + Shift + N**
→ 会在**当前选中目录下**新建文件夹
→ 输入名字回车即可

这是 VS Code **真·默认快捷键**，没有编造。

---

# 2. 在当前目录下新建文件（真实快捷键）
选中目标文件夹后，按：
### **Ctrl + N**
新建空白文件，再按
### **Ctrl + S**
保存时它会自动保存在**当前选中的目录下**。

这也是 VS Code **真实默认**。

---

# 3. 最稳、最不会错的通用方法（100% 真）
不管什么版本都能用：

1. 聚焦文件树：`Ctrl + Shift + E`
2. 选中目标文件夹
3. 按下
## **Alt + Shift + F** 不，是按下：
## **Enter** 打开文件夹
然后再：
- 新建文件：`Ctrl + N` → `Ctrl + S`
- 新建文件夹：`Ctrl + Shift + N`

---

# 4. 折叠 / 展开文件夹（真正能实现的按法）
你之前按了没用，是因为**按法不对**，不是假的。

正确按法（依次按，不是一起按）：
1. 聚焦文件树
2. 按
**Ctrl + K**
（按下松开）
3. 再按
**0**（数字零）
→ 全部折叠

2. 按
**Ctrl + K**
（松开）
3. 再按
**J**
→ 全部展开

这个 **100% 真实有效**。
