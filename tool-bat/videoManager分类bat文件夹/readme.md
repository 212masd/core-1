.# VideoManager 分类批处理工具

## 文件夹概览

**VideoManager 分类批处理工具**是一个用于自动分类和整理视频文件的批处理脚本集合，位于 `d:\b战\BBDown_1.6.3_1080p\core\VideoManager\videoManager分类bat文件夹` 目录下。

### 包含文件

- **main.bat**：主脚本，负责遍历所有子文件夹并执行分类操作
- **reNameVideo.bat**：视频分类脚本，将视频文件移动到以第一个视频文件名命名的文件夹中
- **Filere.bat**：目录迁移脚本，将深层子目录迁移到当前目录

## 功能说明

### 1. 视频自动分类

**功能**：将每个文件夹中的视频文件自动分类到以第一个视频文件名命名的子文件夹中。

**支持的视频格式**：
- MP4
- MKV
- MOV
- AVI
- FLV
- WMV
- RMVB
- MPEG
- MPG

### 2. 批量处理

**功能**：自动遍历当前目录下的所有子文件夹，对每个文件夹执行分类操作。

**处理流程**：
1. 复制 `reNameVideo.bat` 到每个子文件夹
2. 进入子文件夹执行分类操作
3. 退回上一级目录继续处理

### 3. 目录结构整理

**功能**：将深层嵌套的子目录迁移到当前目录，简化目录结构。

**处理流程**：
1. 收集所有深层子目录路径
2. 过滤掉当前目录的直接子目录
3. 将深层子目录迁移到当前目录
4. 清理临时文件

5. 对每个子文件夹：
   - 复制 `reNameVideo.bat` 到子文件夹

6. 执行 `Filere.bat` 整理目录结构

### 视频分类脚本 (reNameVideo.bat)

```batch
@echo off
setlocal enabledelayedexpansion
cd /d "%~dp0"
cd /d
echo 未找到视频文件
pause
:: 找到第一个视频文件（你可以自己加格式）
for %%f in (*.mp4 *.mkv *.mov *.avi *.flv *.wmv *.rmvb *.mpeg *.mpg) do (
set "first_video=%%~nf"
goto :found
)
echo 未找到视频文件
pause >nul
exit

:found
echo 第一个视频：!first_video!
echo 新建文件夹：!first_video!

if not exist "!first_video!" md "!first_video!"

:: 把所有视频移进去
move *.mp4 "!first_video!\" 2>nul
move *.mkv "!first_video!\" 2>nul
move *.mov "!first_video!\" 2>nul
move *.avi "!first_video!\" 2>nul
move *.flv "!first_video!\" 2>nul
move *.wmv "!first_video!\" 2>nul
move *.rmvb "!first_video!\" 2>nul
move *.mpeg "!first_video!\" 2>nul
move *.mpg "!first_video!\" 2>nul

echo 完成！所有视频已移入文件夹：!first_video!
```

**工作流程**：
1. 启用延迟变量扩展
2. 切换到脚本所在目录
3. 查找第一个视频文件
4. 以第一个视频文件的名称创建文件夹
5. 将所有视频文件移动到新创建的文件夹中

### 目录迁移脚本 (Filere.bat)

```batch
@echo off
chcp 65001 >nul 2>&1
title 仅迁移子目录层级 - bite命令
echo ==============================================
echo 【bite命令开始执行】核心：仅迁移子目录，不移动任何文件
echo ==============================================

:: 核心逻辑：先收集所有深层子目录路径 → 过滤掉当前目录的直接子目录 → 逐个迁移到当前目录
echo 【步骤1】定义系统临时文件，用于存储所有深层子目录路径
set "temp_list=%temp%\bite_dir_list.txt"
echo 临时文件路径：%temp_list%

echo 【步骤2】清空临时文件（避免残留旧数据）
echo. > "%temp_list%"

echo 【步骤3】递归查找当前目录下所有子目录（含深层嵌套），仅输出目录路径
echo 正在查找所有深层子目录...
dir /AD /S /B .\* >> "%temp_list%"

echo 【步骤4】遍历所有找到的深层子目录路径，开始迁移
for /f "delims=" %%p in (%temp_list%) do (
    echo 【遍历到目录】完整路径：%%p
    echo 【判断逻辑】检查是否为当前目录的直接子目录（避免重复处理）
    if not "%%~dp"=="%~dp0" (
        echo 【待迁移目录】非直接子目录，准备移动到当前目录：%%p
        echo 【执行移动】move "%%p" .
        move "%%p" . >nul 2>&1
        if errorlevel 1 (
            echo 【迁移失败】目录 %%p 可能已存在/权限不足！
        ) else (
            echo 【迁移成功】目录 %%p 已移至当前目录
        )
    ) else (
        echo 【跳过处理】%%p 是当前目录的直接子目录，无需迁移
    )
)

echo 【步骤5】删除临时文件，清理缓存
del /Q "%temp_list%" >nul 2>&1
echo 临时文件已清理：%temp_list%

echo ==============================================
echo 【bite命令执行完成】所有深层子目录已迁移到当前目录（文件未动）
echo ==============================================
pause >nul
```

**工作流程**：
1. 设置命令行编码为 UTF-8
2. 定义临时文件用于存储目录路径
3. 递归查找所有深层子目录
4. 遍历找到的子目录，将非直接子目录迁移到当前目录
5. 清理临时文件

## 使用方法

### 基本使用

1. **准备工作**：
   - 将需要分类的视频文件放入不同的子文件夹中
   - 确保 `MakerFile.bat` 文件存在（如果不存在，可能需要创建）

2. **执行分类**：
   - 双击运行 `main.bat` 文件
   - 脚本会自动遍历所有子文件夹并执行分类操作

3. **查看结果**：
   - 每个子文件夹中的视频会被分类到以第一个视频文件名命名的子文件夹中
   - 深层子目录会被迁移到当前目录

- 文件夹 `动漫` 包含多个视频文件：

  - `进击的巨人 S01E02.mp4`

- 如果 `动漫` 文件夹在深层目录，会被迁移到当前目录

## 故障排除

### 常见问题

1. **未找到视频文件**：
   - 检查文件夹中是否有支持的视频文件
   - 检查视频文件扩展名是否正确

2. **迁移失败**：
   - 检查目录是否已存在
   - 检查权限是否足够

3. **脚本执行卡住**：
   - 检查是否有文件被其他程序占用
   - 检查目录结构是否过于复杂

### 解决方案

1. **确保文件格式正确**：
   - 确认视频文件使用支持的扩展名
   - 避免使用过长或特殊的文件名

2. **检查权限**：
   - 以管理员身份运行脚本
   - 确保目标目录有写入权限

3. **简化目录结构**：
   - 避免过多的深层嵌套目录
   - 确保目录名称不重复
